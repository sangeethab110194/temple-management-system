# temple-management-system
A web application to manage temple pooja lists, reports and admin operations using React, Tailwind CSS, Flask and SQLite.
Description 
Temple Management Sysytem is a full-stack web application developed to manage temple activities such as pooja list management, reports and adminstation.

Technologies Used:
React
Tailwind CSS
Python
Flask
SQLite

Features:
Admin Login
Manange poojalist
Generate Reports
Simple and user-friendly interface

Project Structure
backend-Flask API and database
frontend-React user interface
